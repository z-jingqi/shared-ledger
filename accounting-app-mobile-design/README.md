# Accounting App Mobile Design Export

本压缩包包含移动端记账产品的设计参考文件，用于交给 AI 或前端开发继续实现。

## 目录

- `screens/`：所有页面的单独 PNG 设计图。
- `docs/tokens.css`：Tailwind v4 + OKLCH 主题变量。
- `docs/screens.md`：页面用途和业务说明。
- `docs/components.md`：基础组件和样式说明。
- `docs/manifest.json`：页面文件索引。
- `assets/account_style_reference.png`：账号页风格参考图。

## 注意

- 每个页面都是独立图片，不是拼图。
- AI Provider 与模型选择不在用户界面中出现，属于后端配置。
- 免费用户没有 AI 助手入口，仅在导入流程中使用后台智能识别。
