## Usage

第一轮建议先评审：

- 底部导航和中间添加按钮是否符合预期。
- 首页信息密度是否合适。
- 账本切换器是否足够轻量。
- 记录表单字段和弹窗是否清晰。
- 全局文件处理状态是否适合在所有页面出现。
- AI 助手是否更像全局助手，而不是独立页面。
